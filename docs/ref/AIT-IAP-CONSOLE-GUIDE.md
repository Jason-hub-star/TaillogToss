## 6. 등록 후 처리

### product_id 연결 확인

콘솔에서 발급된 SKU가 아래 코드의 `product_id`와 **정확히 일치**해야 함:

```typescript
// src/types/subscription.ts
'pro_monthly'   // 비소모성
'ai_token_10'   // 소모성
'ai_token_30'   // 소모성
```

### Backend verify-iap-order Edge Function 확인

```
supabase/functions/verify-iap-order/index.ts
→ orderId 검증 후 subscriptions 테이블 업데이트
→ PRO: plan_type = 'PRO_MONTHLY'
→ 토큰: ai_tokens_remaining += 10 or 30
```

### 상품 이미지 규격

- 1024 × 1024 px
- 배경: 단색 (앱스토어 심사 기준)
- 권장: 테일로그 로고 + 상품명 텍스트
